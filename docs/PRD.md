# 產品需求文件（PRD）— 個人記帳簿系統

## 1. 專案概述

### 1.1 背景與動機

在日常生活中，許多人對自己的收入與支出缺乏系統性的追蹤，導致無法掌握個人財務狀況。雖然市面上有許多記帳 App，但對於只想簡單記錄、快速瀏覽報表的使用者來說，功能過於複雜。本專案旨在打造一個**簡單、直覺、輕量**的個人記帳簿網頁系統，讓使用者能夠快速記錄每日收支，並透過統計報表清楚掌握自己的財務狀況。

### 1.2 目標用戶

- 想要養成記帳習慣的一般使用者
- 需要追蹤個人收入與支出的學生、上班族
- 希望透過簡單工具了解自己花費狀況的人

### 1.3 核心價值主張

- **簡單易用**：介面簡潔，操作流程直覺，降低記帳門檻
- **即時統計**：自動計算總花費與餘額，無需手動加總
- **視覺化報表**：透過圖表呈現財務趨勢，幫助使用者做出更好的理財決策

## 2. 功能需求

### 功能一：記錄收入

> 作為一個**記帳使用者**，我希望能**新增一筆收入記錄（包含日期、金額、分類、備註）**，以便**追蹤我的所有收入來源**。

- 使用者可輸入：日期、金額、分類（如：薪資、獎金、兼職、投資收益、其他）、備註
- 預設日期為當天
- 金額必須為正數
- 新增成功後顯示確認訊息

### 功能二：記錄支出

> 作為一個**記帳使用者**，我希望能**新增一筆支出記錄（包含日期、金額、分類、備註）**，以便**了解我的錢花到哪裡去了**。

- 使用者可輸入：日期、金額、分類（如：餐飲、交通、娛樂、購物、日用品、醫療、教育、其他）、備註
- 預設日期為當天
- 金額必須為正數
- 新增成功後顯示確認訊息

### 功能三：每日收支明細

> 作為一個**記帳使用者**，我希望能**依照日期瀏覽每天的收入與支出明細**，以便**回顧當天的財務活動**。

- 以日期為單位，列出當天所有收入與支出記錄
- 支援按日期篩選（日曆選擇器）
- 每筆記錄顯示：類型（收入/支出）、分類、金額、備註
- 可編輯或刪除個別記錄

### 功能四：總花費與餘額統計

> 作為一個**記帳使用者**，我希望能**一目瞭然地看到總收入、總支出與餘額**，以便**快速掌握目前的財務狀況**。

- 首頁儀表板顯示：
  - 總收入
  - 總支出
  - 結餘（總收入 − 總支出）
- 支援依月份篩選統計數據
- 數字即時更新（新增/編輯/刪除記錄後自動刷新）

### 功能五：財務報表

> 作為一個**記帳使用者**，我希望能**透過圖表檢視支出分類比例與收支趨勢**，以便**分析我的消費習慣並調整理財策略**。

- 支出分類圓餅圖：顯示各分類佔總支出的比例
- 收支趨勢折線圖：顯示每日或每月的收入與支出變化
- 支援選擇時間範圍（本週、本月、自訂區間）
- 報表可依月份切換檢視

## 3. 非功能需求

### 3.1 技術限制

| 項目         | 技術選擇                     |
| ------------ | ---------------------------- |
| 後端框架     | Flask (Python)               |
| 模板引擎     | Jinja2                       |
| 資料庫       | SQLite                       |
| 前端         | HTML + CSS + JavaScript      |
| 圖表庫       | Chart.js（前端繪圖）         |

### 3.2 效能考量

- 頁面載入時間應在 **2 秒以內**
- 資料庫查詢應做適當索引，避免大量記錄時效能下降
- 報表圖表應在資料載入完成後 **1 秒內** 完成渲染

### 3.3 安全考量

- 所有使用者輸入需進行**輸入驗證與消毒**（防止 SQL Injection、XSS）
- 金額欄位僅接受數字輸入
- 日期欄位需驗證格式正確性

### 3.4 使用性考量

- 介面需支援**響應式設計（RWD）**，在手機與桌面瀏覽器皆可正常使用
- 操作流程不超過 **3 步** 即可完成一筆記帳
- 提供清楚的錯誤訊息與操作回饋

## 6. 附錄

### 頁面清單（初步規劃）

| 頁面名稱       | 路由             | 說明                         |
| -------------- | ---------------- | ---------------------------- |
| 首頁（儀表板） | `/`              | 顯示總收入、總支出、結餘     |
| 新增記錄       | `/add`           | 新增收入或支出的表單頁面     |
| 每日明細       | `/daily`         | 依日期檢視收支明細           |
| 編輯記錄       | `/edit/<id>`     | 編輯單筆收支記錄             |
| 財務報表       | `/report`        | 圖表化的財務分析報表         |

### 資料欄位（初步規劃）

| 欄位名稱   | 類型     | 說明                                 |
| ---------- | -------- | ------------------------------------ |
| id         | INTEGER  | 主鍵，自動遞增                       |
| type       | TEXT     | 類型：`income`（收入）/ `expense`（支出） |
| amount     | REAL     | 金額                                 |
| category   | TEXT     | 分類名稱                             |
| note       | TEXT     | 備註（可為空）                       |
| date       | DATE     | 記錄日期                             |
| created_at | DATETIME | 建立時間                             |
